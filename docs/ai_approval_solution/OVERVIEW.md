# 文件夹说明

这个目录用于整理“AI 智能审批助手”的具体落地方案。

## 文件

- `README.md`：主方案文档，包含场景目标、LangGraph 节点设计、模型选择、向量数据库、接口清单、RAG 知识库、回退修改、安全风控和落地计划。

## 建议后续可补充

- `api_contract.md`：审批系统接口字段定义。
- `graph_design.md`：LangGraph 节点和边的详细设计。
- `prompt_design.md`：意图识别、字段抽取、审批预览等 prompt。
- `evaluation.md`：试点评估指标和测试用例。
- `demo_plan.md`：Demo 开发计划。
