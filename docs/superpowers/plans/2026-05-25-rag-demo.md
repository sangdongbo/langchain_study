# Streamlit RAG Demo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an isolated Streamlit RAG demo that indexes uploaded txt/markdown files into Chroma and answers questions with DeepSeek using LCEL.

**Architecture:** Keep RAG mechanics in `rag_demo/rag_chain.py` and UI in `rag_demo/app.py`. Persist vector data to `rag_demo/chroma_db/`, and use pure helper tests that avoid Chroma writes and model calls.

**Tech Stack:** Python, Streamlit, LangChain LCEL, ChromaDB, DeepSeek through `ChatOpenAI`, unittest.
