# 健康饮食基础

健康饮食的核心是食物多样、总量适度、长期稳定。日常饮食可以优先选择全谷物、蔬菜、水果、豆类、鱼禽蛋奶和适量坚果，减少高盐、高糖、高油食物。

## 每日饮食建议

- 主食可以粗细搭配，例如糙米、燕麦、玉米、红薯和普通米饭轮换。
- 每天尽量吃足量蔬菜，深色蔬菜可以占一半左右。
- 水果适合作为加餐，但果汁不能完全替代新鲜水果。
- 蛋白质来源可以包括鸡蛋、牛奶、鱼肉、鸡肉、豆腐和豆浆。
- 饮水优先选择白水或淡茶，少喝含糖饮料。

## 控盐控糖

成年人日常饮食应注意减少盐、酱油、咸菜、加工肉类和高糖点心。长期高盐饮食可能增加血压负担；长期高糖饮食可能增加体重管理难度。
