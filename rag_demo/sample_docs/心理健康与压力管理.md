# 心理健康与压力管理

心理健康和身体健康一样重要。压力本身并不一定有害，但长期处于高压状态，可能影响睡眠、食欲、注意力、人际关系和工作效率。

## 常见压力信号

- 入睡困难、早醒或睡眠质量下降。
- 容易烦躁、焦虑、情绪低落或失去兴趣。
- 注意力下降，工作和学习效率变差。
- 头痛、胃部不适、肌肉紧张等身体反应。
- 回避社交或对日常事务感到明显疲惫。

## 日常调节方法

- 把任务拆小，先完成最容易开始的一步。
- 每天安排短时间活动，例如散步、拉伸或深呼吸。
- 保持稳定作息，减少熬夜。
- 和可信任的人交流，不把压力长期憋在心里。
- 给自己设置休息边界，避免一直处于工作状态。

## 需要寻求帮助的情况

如果持续两周以上明显情绪低落、焦虑严重、无法正常生活，或出现伤害自己或他人的想法，应尽快联系专业心理咨询师、精神科医生或当地危机干预资源。

本文件仅用于科普和测试，不替代专业心理健康服务。
