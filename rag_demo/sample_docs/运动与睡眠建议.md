# 运动与睡眠建议

规律运动和充足睡眠是维持健康状态的重要基础。对于多数成年人来说，适量有氧运动、力量训练和稳定作息可以帮助改善体能、情绪和日间精力。

## 运动建议

- 每周可以安排 3 到 5 次中等强度有氧运动，例如快走、骑车、游泳或慢跑。
- 力量训练可以每周 2 到 3 次，重点训练腿部、背部、胸部、肩部和核心肌群。
- 久坐人群建议每 45 到 60 分钟起身活动 3 到 5 分钟。
- 初学者应从低强度开始，逐步增加时间和频率。

## 睡眠建议

- 尽量保持固定入睡和起床时间。
- 睡前减少咖啡、浓茶、酒精和长时间刷手机。
- 卧室环境保持安静、昏暗、温度适宜。
- 如果长期严重失眠、白天嗜睡或打鼾明显，应考虑咨询专业医生。

## 注意事项

如有心血管疾病、严重关节问题、孕期或近期手术史，开始新的运动计划前应先咨询医生。
